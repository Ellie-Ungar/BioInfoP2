Class Name: LocalAlignmentAlgorithm

Content: This class contains four algorithms to perform alignment scoring for two nucleotide strings being compared. 
The four algorithms are:
1. The Smith-Waterman
2. Divide and Conquer
3. Greedy
4. Brute Force

Inputs and Outputs: This class is not intended to accept inputs and provide outputs independantly.
It is intended to be used in conjunction with the GUI class, which will take in user inputs, call this class, and provide outputs to the user.
